列表是很常见的，横向菜单栏，竖向菜单栏，新闻列表之类的，还会说到盒子定位中的另一个属性position，后面会具体补充说这个属性，以及设置元素堆叠顺序的z-index属性。

列表

list-style-type

设置列表项标记类型，主要有以下几种

  • none 无标记符号
  • disc 实心圆（默认）
  • circle 空心圆
  • square 实心正方形
  • decimal 数字

list-style-image

使用图像来替代上面说到的标记，设置image图像之后，list-style-type属性将不再起作用，实际使用的时候要避免某些浏览器不支持这个属性，还是要设置list-style-type这个属性。

list-style-position

设置何处放置列表项的标记，有两个属性值，inside和outside。
inside表示项目标记放置在文本以内，且环绕文本根据标记对齐。
outside默认值，表示标记位于文本左侧，列表项标记放置在文本以外，且环绕文本不根据标记对齐。
但是我并没有发现有什么区别，至少相对于普通文字列表来说。
