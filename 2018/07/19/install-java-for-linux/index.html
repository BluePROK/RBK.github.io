有了服务器之后当然就要装点有用的东西了，比如我在linux使用Java语言，比如我之前玩的socket通信，这次先说安装Java的环境，版本就jdk8吧，by the way，我之前不是看了lambda表达式吗？我本地jdk6不支持，顺便在这也能用上了，话说我应该也得考虑把我的jdk6升级了。再说变说几个常用的linux命令，之后再说说关于装其他东西的文章，比如微皮恩之类的。

使用工具和环境

  • xshell 5（Windows7系统）
  • Ubuntu16.04（Linux服务器）
  • jdk-8u181-linux-x64.tar.gz（点击此处进入下载页面）

linux常用的命令

此处我都没有带参数，具体使用详见百度（至少知道该百什么）

  • 下载文件 wget [文件名]
  • 删除文件 rm [文件名]
  • 新建文件夹 mkdir [文件夹名]
  • 复制文件 cp [指定文件目录/文件名] [目标目录/新文件名]
  • 移动文件 mv [指定文件目录/文件名] [目标目录/新文件名]（mv命令也能重命名文件，参数即为[旧文件名 新文件名]）
  • 查看文件 ls（参数-lht查看详细信息）
  • 当前目录 pwd
