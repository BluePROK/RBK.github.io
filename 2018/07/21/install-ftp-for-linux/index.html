到此FTP服务安装完成，下面进行Web服务器的配置。

安装Web服务器相应程序

MySQL5

sudo apt-get install mysql-server mysql-cient

Apache2

Apache2作为Ubuntu的一个软件包，可以用命令直接安装。
sudo apt-get install apache2
此时使用浏览器访问127.0.0.1，应该能看到Apache2的测试页面，不过服务器上好像看不到…
默认文档根目录是Ubuntu的/var/www目录，配置文件是/etc/apache2/apache2.conf，配置存储在子目录/etc/apache2。
Apache默认只支持Html，通过插件可以支持PHP页面，还可以与Tomcat进行连接，后面后端我可能会用到，这是单向连接，只能由Apache访问Tomcat，反之则不可以。

PHP

可以直接安装PHP7.0和Apache的PHP7.0模块，也可以选择PHP5版本，但是我好像找不到这个版本，就换了一个。
sudo apt-get install php7.0 libapache2-mod-php7.0
安装完成之后需要重启/etc/init.d/apacne2 restart

至此Apache2+PHP7+MySQL5的服务器环境就配置完成了，此时若在主机中访问127.0.0.1或者外网访问IP地址（如我的：45.199.183.68）便可看到Apache2的测试页面，即安装成功。
